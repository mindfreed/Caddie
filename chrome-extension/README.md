# Freedmind AI Caddy - Chrome Extension

The Caddy floats on ANY website as a transparent overlay. True saran-wrap mode.

## Installation (Developer Mode)

1. Download this `chrome-extension` folder to your computer
2. Open Chrome and go to: `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `chrome-extension` folder
6. Done - the Caddy now floats on every site you visit

## How to Use

1. Go to any offer wall (Freecash, KashKick, Swagbucks, etc.)
2. Copy a game name to your clipboard
3. Tap the faded Caddy icon in the bottom-right corner
4. He pops up, scans your clipboard, and gives you the intel
5. Enter the offer amount to get custom EV calculation

## Features

- Floats on ANY website
- Reads clipboard for game names
- Cross-references Alliance Stats database
- Custom offer amount calculator
- Tactical wingman voice
- Red alert for high-risk games
